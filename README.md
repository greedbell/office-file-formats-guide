# office-file-formats-guide

老版 Office 文件（doc, xls, ppt）格式分析

老版 Office 文件主要分为 5 个部分：

- MS-DOC: Word
- MS-XLS: Excel
- MS-PPT: PPT
- MS-ODRAW: Word、Excel、PPT 内的图表
- MS-OSHARED: Word、Excel、PPT 共享的
